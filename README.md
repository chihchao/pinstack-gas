# Pinstack 極簡看板平台

一個以 **Google Apps Script + Google Sheets** 打造的輕量版 Pinstack 看板分享平台。免登入、免資料庫、免主機費用，資料全部存放在你自己的 Google 試算表與雲端硬碟中。

## 功能介紹

- **看板（Page）管理**：建立多個看板，每個看板可獨立設定標題、描述；管理員可將看板設為「關閉」，關閉後一般使用者無法再新增/編輯貼文（但仍可瀏覽）。
- **貼文（Post）互動牆**：瀑布流卡片版面，支援文字內容、參考網址、上傳圖片（前端自動壓縮至 300KB 以下）。
- **網址連結預覽**：貼文中的參考網址會透過 [Microlink](https://microlink.io/) API 自動抓取標題、描述、縮圖，產生預覽卡片。
- **按讚與留言**：每篇貼文可按讚計數，並可展開詳細視窗留言討論。
- **免登入暱稱機制**：使用者首次進站只需設定一個顯示暱稱（存於瀏覽器 `localStorage`），之後可隨時更換；貼文/留言的「擁有者」判斷即以暱稱比對。
- **管理員模式**：輸入共用管理員密碼後可新增/編輯/刪除看板，並可編輯或刪除任何人的貼文；密碼存放於 Apps Script 的「指令碼屬性」，不寫死在程式碼中。
- **看板分享連結**：在看板內頁點擊「複製分享連結」，會產生 `{Web App 網址}?page=看板id` 格式的網址；他人開啟該網址時會自動載入並直接切換到該看板。

## 系統架構

- `Code.gs`：後端邏輯，透過 `google.script.run` 供前端呼叫，資料讀寫皆對應同一份 Google 試算表中的 `pages`、`posts`、`messages` 三個工作表；圖片上傳則寫入指定的 Google 雲端硬碟資料夾。
- `Index.html`：整個前端頁面（HTML + Tailwind CDN + Font Awesome CDN + 原生 JavaScript），由 `doGet()` 以樣板方式渲染輸出。

## 安裝需求

- 一個 Google 帳號（可存取 Google 試算表 / Apps Script / 雲端硬碟）
- 不需要任何本機開發環境、Node.js 或套件安裝

## 安裝步驟

### 1. 建立 Google 試算表並設定工作表

新增一份 Google 試算表，並建立以下三個工作表（分頁名稱需完全一致），第一列為表頭：

| 工作表名稱 | 表頭欄位（由左至右） |
|---|---|
| `pages` | `id`, `title`, `description`, `datetime`, `closed` |
| `posts` | `id`, `page_id`, `content`, `url`, `image`, `poster`, `datetime`, `likes` |
| `messages` | `id`, `post_id`, `message`, `messager`, `datetime` |

> 欄位順序很重要：後端部分邏輯以欄位索引直接讀寫（例如 `posts` 的 `poster` 需固定在第 6 欄 F），請勿自行調整欄位順序。

### 2. 建立 Apps Script 專案

在試算表中點選「擴充功能」→「Apps Script」，開啟綁定的指令碼專案：

1. 將本專案的 `Code.gs` 內容貼入預設的 `Code.gs` 檔案。
2. 新增一個 HTML 檔案，命名為 `Index`（副檔名 `.html` 由編輯器自動附加），貼入本專案的 `Index.html` 內容。

### 3. 建立圖片上傳資料夾

在 Google 雲端硬碟建立一個資料夾（用來存放使用者上傳的貼文圖片），並複製其資料夾 ID（網址列中 `folders/` 後面那段字串）。

### 4. 設定指令碼屬性

在 Apps Script 編輯器左側「專案設定」→「指令碼屬性」，新增以下兩筆屬性：

| 屬性名稱 | 說明 |
|---|---|
| `ADMIN_PASSWORD` | 管理員登入用的共用密碼 |
| `FOLDER_ID` | 步驟 3 建立的雲端硬碟資料夾 ID |

### 5. 部署為網頁應用程式

點選「部署」→「新增部署作業」：

- 類型選擇「網頁應用程式」
- 執行身份：**我**（以擁有者身份執行，前端才能讀寫試算表與雲端硬碟）
- 存取權限：依需求選擇「所有人」（公開分享）或「知道連結的使用者」

部署完成後，複製結尾為 `/exec` 的網址，即為看板平台的公開網址。

## 使用方式

1. 首次開啟網址時，會要求設定顯示暱稱（僅存於當前瀏覽器）。
2. 首頁列出所有看板，點擊卡片即可進入該看板瀏覽貼文。
3. 在看板內頁點擊「發表新貼文」可張貼文字、網址或圖片；點擊「複製分享連結」可取得可直接開啟該看板的網址（`?page=看板id`）。
4. 點擊右上角「管理員模式」並輸入 `ADMIN_PASSWORD` 後，可新增/編輯/關閉/刪除看板，並管理所有貼文。
